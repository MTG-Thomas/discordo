# Security Policy

Discordo handles Discord authentication tokens and stores saved tokens in the operating system keyring.

Do not open a public issue for a vulnerability that could expose tokens, local files, or Discord account access. Report upstream security concerns to the upstream project maintainer:

https://github.com/ayn2op/discordo/security

For fork-specific concerns in `MTG-Thomas/discordo`, contact the repository owner privately.

## Supported Versions

This fork tracks the upstream `main` branch. Security fixes should be based on the latest upstream code whenever possible.
